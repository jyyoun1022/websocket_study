###  웹소켓 이란?
- 웹소켓은 서버와 클라이언트 사이에 소켓 커넥셔을 유지하면서, 양빵향 통신이 가능한 기술이다.

- 웹소켓은 HTTP로 Handshake를 초기 통신을 시작한 후, 웹소켓 프로토콜로 변환하여 데이터를 전송한다.

> Handshake 
> >클라이언트와 서버 간의 연결을 확립하는 초기 단계로, 웹소켓 프로토콜이 HTTP(S)를 통해 수행되는 일반적인 통신에서 전환되는 과정
